# StylerNow Documentation Hub

6. [DEV_GUIDE.md](./DEV_GUIDE.md) - Guía interna para desarrolladores, convenciones de código, monorepo setup y branching model.
7. [CHANGELOG.md](./CHANGELOG.md) - Historial de versiones y modificaciones mayores de producto.
8. [SECURITY_GUIDE.md](./SECURITY_GUIDE.md) - Políticas Zero Trust Fintech, configuración RLS, autenticación multifactor (si aplica) y normativas OWASP.
